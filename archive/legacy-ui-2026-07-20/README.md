# Legacy StringSight UI archive — 2026-07-20

This directory preserves the presentation experiments that preceded the current tab-centered
practice workspace. They are intentionally excluded from the active application and build.

Archived directions:

- `PracticeWorkspace`: paper score with library/inspector layout.
- `PracticeWorkspaceV2`: marked-passage concept.
- `PracticeWorkspaceV3`: dense video/audio control-room concept.
- `RackWorkspace` and `ui/rack`: hardware-rack design system used by the original audio workbench.
- The rack and mockup styles listed in this directory. The former global stylesheet is retained in
  Git history rather than duplicated here.

The reusable audio, music, session, persistence, and worker controllers were not archived. The new
workspace consumes those functional modules directly.

The approved standalone design reference remains at
`mockups/stringsight-dual-canvas.html` and is not part of the production application bundle.
